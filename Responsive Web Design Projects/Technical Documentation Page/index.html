What is R?

R is an free and open-source programming language that is used for statistical computation. Some examples of industries that utilize the R language include:

  • Data Mining
  • Academic Research
  • Financial Services

Here is a simple example of what R looks like. We utilize the print function of R to make our computer say "Hi" to us (and to the whole world!).

x <- "Hello World" print(x) //this will print out "Hello World"

Below is an introduction by Basecamp to what R is like. Watch the video in order to get the basics before we begin our journey into R.

Obtaining R

To get started, we need to obtain two things: R and RStudio. Follow these instructions.

  1. Install R by using a CRAN mirror, https://cran.r-project.org/mirrors.html where you will get the R console onto your computer.
  2. Go to https://www.rstudio.com/ to get your R workspace (this is a very helpful tool that you should utilize for all your R projects).
  3. Make sure that you have all of the programs up to date. Alternatively, you can use a program such as https://code.visualstudio.com/ or even codepen.io in order to code in R.

VaRiables

Like every programming language, R has a distinct way in order to create variables. A variable is a storage location that is associated with a symbolic name that allows the programmar to obtain and remember the value in a much simpler format. For instance:

x <- 4:23 # This allows us to create a complex numeric vector utilizing numbers that we typically do not tabulate. y <- x^4 # A vector will be formed based on the values that we get from x. print(y) # The print function will allow us to print the value of the vector we created. We'll talk more about functions in the next section. # Output
[1] 256 625 1296 2401 4096 6561 10000 14641 20736 28561 [11] 38416 50625 65536 83521 104976 130321 160000 194481 234256 279841

Even in simple operations, we can utilize variables that will allow us to get the value that we are looking for. Variables will become much more important the more data analysis we will have to do. Take from example a part of a code taken in a Life Sciences statistical analysis class that I have taken

obs_fit <- lm(temp ~ spot, data = ireland) slopes <- numeric(500) set.seed(572) for (i in 1:2000){ hrtrate_shuffle_i <- sample(ireland$hrtrate) fit_i <- lm(temp ~ spot_shuffle_i, data = ireland) slopes[i] <- fit_i$coefficients[2] } mean(slopes >= obs_fit$coefficients[2] | slopes <= -obs_fit$coefficients[2] ) # Output
[1] 0.0025

We can see how we created two variables, obs_fit and slopes in order to insert them into a for loop with different mathematical expressions and logical arguments to get our value of 0.0025. While in the beginning it may seem difficult to utilize variables, it is important to get the logic behind the creation of such tools in order to assist you on the R journey in front of you!

R Functions

There are other functions in R that will help you for the beginning. Play around with them as you delve deeper into R. Remember, R is your playground, and you are allowed to utilize it in whatever way you like!

While it is technically possible to insert the dataset as is with the following functions, it is good practice to be able to turn the datasets into variables instead. When working with multiple lines of code, this will be more clear and less mistakes will be made!

Getting A Package

The wonderful aspect of R being free is that there is a huge open-source community that constantly creates new features and functionalities that is not included in the R language. In order to implement these new packages, we utilize the "library" function.

library(x)

It's important that you utilize this function whenever you start a new project. Everytime you open a new project, the settings of your R workspace may default back to the basic R functions. Therefore, a good practice to have is to start with the "library" function.

Finding the Total Sum

Perhaps that you have a dataset that you just need to find the total sum of. If you express the dataset as a variable (for our example, we will utilize "x"), R will be able to tell you the total sum of the dataset!

sum(x)

Finding Linear Correlation

Perhaps you're a data scientist and you want to find a linear correlation between two datasets that are expressed in two variables. These datasets can also be matrices or data frames. In all cases, you can utilize the "cor" function to find the linear relationship (or a correlation matrices if it's data frames or matrices) between the two variables.

cor(x, y)

Expressing Datasets on a 2-D X-Y graph

Let's take it one step further and plot the linear relationship on a 2-D graph. R allows you to do that by using the "plot" function. Simply express the datasets as two variables and then you are able to use the function.

plot(x, y)

Plotting Histograms

We can plot histograms in R by using the "hist" function. We first need to define a dataset by using a variable. Once we obtain the variable, we then use the "hist" function.

hist(x)
